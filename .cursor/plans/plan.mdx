## Phase 1: Foundation Enhancement (Week 1-2)

### 1.1 Schema Markup Enhancement
**Goal:** Add alternate names to structured data so Google recognizes all spelling variations

**Actions:**
- Update Organization schema in `Layout.astro` to include `alternateName` array:
  - "Janaki Mahal Trust"
  - "Sri Janki Mahal Trust" 
  - "Shri Janki Mahal Trust"
  - "Janki Mahal Trust Ayodhya"
- Add same alternate names to LocalBusiness schema
- Add same alternate names to LodgingBusiness schema

**Why:** Google will associate all these variations with your business entity

### 1.2 Image Rich Snippets Implementation
**Goal:** Make images appear in Google Search results below your listing

**Actions:**
- Add ImageObject schema to homepage with key images (rooms, exterior, temple)
- Create Gallery schema on dedicated gallery page with structured markup
- Add Product/Offer schemas with images for each room type
- Implement image sitemap (separate XML file listing all important images with captions, titles, and geo-location)
- Add proper `width`, `height`, and `alt` attributes to all important images
- Create caption-rich image markup using `<figure>` and `<figcaption>` HTML5 tags

**Why:** Google displays image carousels when proper schema and markup exists

### 1.3 SEO Utility Updates
**Goal:** Support all keyword variations in meta tags system

**Actions:**
- Update `src/lib/seo.ts` DEFAULT_KEYWORDS to include all variations
- Add keyword variation generator function that creates combinations
- Ensure every page includes primary keyword + all variations

**Why:** Broader keyword coverage without keyword stuffing

## Phase 4: Image SEO Implementation (Week 4-5)

### 4.1 Image Sitemap Creation
**Actions:**
- Create `/public/image-sitemap.xml`
- List all important images:
  - Room images (all 8 room types, 3-4 images each)
  - Exterior shots (trust building, entrance)
  - Interior common areas
  - Location/landmark images
- For each image include:
  - `<image:loc>` (full URL)
  - `<image:caption>` (descriptive with keywords)
  - `<image:title>` (keyword-rich title)
  - `<image:geo_location>` (Ayodhya, Uttar Pradesh)
  - `<image:license>` (your website URL)

**Why:** Helps Google index images properly for image search and rich results

### 4.2 Image Schema Markup
**Actions:**
- Add ImageObject schema to all room pages
- Add ImageGallery schema to gallery page
- Include in schema:
  - contentUrl
  - caption
  - description
  - creator (Sri Janaki Mahal Trust)
  - copyrightHolder
  - width/height dimensions

**Why:** Rich image results in main search

### 4.3 Structured Image HTML
**Actions:**
- Wrap key images in semantic HTML:
```html
<figure>
  <img src="..." alt="2 Bed AC Room at Janaki Mahal Trust Ayodhya" width="800" height="600">
  <figcaption>Comfortable 2 Bed AC Room at Sri Janki Mahal Trust</figcaption>
</figure>
```
- Add title attributes to images
- Ensure all alt text includes keyword variations naturally

**Why:** Semantic HTML helps Google understand image context

### 4.4 Open Graph Image Optimization
**Actions:**
- Add multiple `og:image` tags (up to 6 images per page)
- Specify different images for room pages, location pages
- Add `og:image:width` and `og:image:height`
- Create dedicated OG images with text overlays for key pages

**Why:** Better social sharing + signals to Google
