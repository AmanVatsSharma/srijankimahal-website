# Ayodhya Check-in Requirements (ID Proof + Basic Rules) — Quick Guide

To avoid last-minute stress at check-in, keep this simple checklist ready.

## ID proof (carry valid government ID)

- Carry valid ID for adult guests.
- Keep the booking confirmation message handy (screenshot).

## What details to keep ready

- Guest names + phone number
- Check-in/check-out dates
- Room type (AC/Non-AC, beds)
