# Ayodhya Disabled Accessibility - Janaki Mahal Wheelchair

Sri Janaki Mahal Trust offers some accessibility for disabled pilgrims. Confirm current facilities with [official contact](/contact-number).

## What to Request

- **Ground floor rooms**: Essential for wheelchair users. See [senior and floor preferences](/blog/janaki-mahal-trust-senior-citizen-room-floor-preferences)
- **Ramps**: Check for building access
- **Attached bathroom**: See [attached bathroom](/blog/sri-janaki-mahal-trust-attached-bathroom-rooms)
- **Lift**: See [lift access](/blog/does-janaki-mahal-have-lift-elevator-access)

See [wheelchair accessibility](/blog/is-janaki-mahal-trust-wheelchair-accessible) and [accessibility features](/blog/sri-janaki-mahal-trust-accessibility-features).

## Booking

Book via [official booking](/official-booking) or [contact number](/contact-number). State accessibility needs clearly.
