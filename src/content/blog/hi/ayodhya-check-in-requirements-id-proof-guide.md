# अयोध्या चेक-इन आवश्यकताएँ (ID Proof + बेसिक नियम) — क्विक गाइड

चेक-इन पर आखिरी समय की परेशानी से बचने के लिए यह चेकलिस्ट तैयार रखें।

## ID प्रूफ (सरकारी पहचान पत्र)

- सभी वयस्क मेहमानों का वैध ID रखें।
- बुकिंग कन्फर्मेशन संदेश/स्क्रीनशॉट सुरक्षित रखें।

## कौन सी डिटेल्स तैयार रखें

- नाम + मोबाइल नंबर
- चेक-इन/चेक-आउट तारीखें
- रूम टाइप (AC/Non-AC, बेड)
