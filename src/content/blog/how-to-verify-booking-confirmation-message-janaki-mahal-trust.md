# How to Verify a Booking Confirmation Message (Sri Janaki Mahal Trust) — Checklist

If you receive a “booking confirmed” message from any number, verify it before paying.

## What a genuine confirmation should include

- Guest name + phone (or booking reference)
- Check-in and check-out dates
- Room type (AC/Non-AC, 2/3/4 bed, suite)
- Total amount and what is included (meals)
- Clear check-in/check-out timing (as applicable)

## What should NOT happen (red flags)

- Different contact number than the official one
- Payment pressure before a clear confirmation
- Requests to pay to personal accounts or unknown links
