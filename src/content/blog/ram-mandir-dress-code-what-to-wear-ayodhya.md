# Ram Mandir Dress Code - What to Wear Ayodhya

Modest, traditional attire is preferred for Ram Mandir darshan.

## Guidelines

- **Men**: Kurta-pyjama, dhoti, or full pants and shirt
- **Women**: Saree, salwar kameez, or modest dress
- **Avoid**: Shorts, sleeveless, revealing clothes
